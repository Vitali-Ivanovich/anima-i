# Telegram Publishing Config

Этот файл читается агентом и скриптом publish_telegram.sh.
Агент должен сформировать пост и вызвать publish_telegram.sh после каждого шага.

## Настройки

- **channel:** @anima_am_i
- **publish_after_each_step:** true
- **publish_journal:** true
- **publish_knowledge:** true

## Формат поста

Агент формирует пост самостоятельно на основе содержимого JOURNAL.md и KNOWLEDGE.md текущего шага.
Пост должен быть читабельным для внешней аудитории — без технических деталей о файлах и скриптах.
Максимум 3000 символов. Если контент длиннее — агент сам решает что важнее.

**Поколение N | Шаг X**
